# RCNNG — Resonance of Closed Neural Network Geometries

This repository contains the full Python codebase, simulation outputs, and conceptual figures
associated with the RCNNG framework.

## Contents

### 1. Python Code
All scripts used for network simulations, Fourier analyses, geometric reconstructions, and
topological analyses are located in the `code/` directory:

- rcnng_network_simulator.py
- single_frequency_crg.py
- single_frequency_crg_base_trajectory.py
- multi_frequency_crg.py
- multi_frequency_geometry.py
- persistent_homology_of_crg_geometry.py
- conceptual_fourier_geometric_expansion.py
- fourier_geometric_expansion_panels.py

### 2. Simulation Figures
Full-resolution simulation figures (8 plots per network size) for all networks:

- N = 50, 80, 100, 200, 500, 2000, 5000

are located in the `figures/` directory.

### 3. Conceptual Figures
Five conceptual figures illustrating the two perceptual formalisms:

- Formalism I: Resonant-Geometric Encoding
- Formalism II: Spectral-Topological Encoding

are located in the `conceptual_figures/` directory.

### 4. Supplementary PDF
The supplementary document submitted with the manuscript is located in:

`/supplementary/supplementary_RCNNG_Open_Mind.pdf`
